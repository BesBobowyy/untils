*mplib* - Is light-weight library for console games or small utils to process user input as commands and describe structure with config.

## Abilities

- Module-based config in markup line.
- Custom logic in OOP.
- Validation and parsing user input for 10 lines.
- Process exceptions.
- Almost-perfect detailed annotations everywhere.
- Fast command processing (~208Âµs).
- Infinitely extendable abilities.
- Calm about anything.
